Environments
------------
1. Install rvm, ruby 1.9.3, create gemset raffe

2. Run bundle to install gems

    $ rvm 1.9.3@raffe

    $ bundle

3. Start app using shotgun (auto redeploy after code change)
  
    $ shotgun

4. Open http://localhost:9393
